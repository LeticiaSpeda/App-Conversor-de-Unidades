# App-Conversor-de-Unidades

Nesse projeto voce pode saber 4 formas de unidade de medida
e converter qualquer numero desejado 
<div>
  São eles:
  <div>
    
  - Temperatura (Celsius/Farenheint)
    
  - Peso (Libra/Kilograma)
    
  - Moedas (Dolár/Real)
    
  - Metros (Kilometro/Metros)
